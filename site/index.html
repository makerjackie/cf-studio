远程 Cloudflare 资源工作台

CFDesk

面向 Cloudflare 日常运维的桌面工作台。管理 R2、D1、Workers、Queues、KV 和 Token 权限时，把高频操作放在一个低干扰界面里。

查看仓库 查看更新

产品定位

为反复出现的 Cloudflare 运维任务而做。

CFDesk 专注远程账号资源：R2 素材、D1 数据、Workers 运维、Queues、KV 和账号权限检查。

需要本地 wrangler dev 绑定浏览时，CFDesk 会把用户引导到 Cloudflare 官方 Local Explorer。

功能范围

覆盖存储、数据、计算和安全检查的高频入口。

01

R2 素材工作流

浏览存储桶，上传和下载文件，预览图片，复制公开 URL，检查公开域名，并在危险对象操作前确认。

02

D1 数据库工作

查看数据库和表，运行 SQL，查看结构图，管理索引，并导出常用格式。

03

Workers 运维

打开 Dashboard，检查设置和部署，查看近期健康信号，调整观测设置，复制常用 Wrangler 命令。

04

权限与安全操作

检查 Token 权限，对删除、覆盖、Secret、路由、域名、计划任务等远程写操作加入确认。

4 月 18 日之后

一个有独立方向的远程运维 fork。

Token 引导

通过 macOS Keychain 保存 API token，检查端点权限，并在 Wrangler 不可用时提供更清晰的回退路径。

R2 升级

加入缓存列表、图片预览、公开 URL 处理、传输设置、分片上传和更安全的变更确认。

Workers 和 Queues

加入 Workers 快捷操作、近期健康信号、指标、观测控制、设置检查，以及 Queue backlog 指标。

中英文界面

主要页面、设置页和资源操作文案进入中英文结构，方便后续持续维护。
