Cloudflare 远程运维桌面端

CFDesk

CFDesk 把 R2 文件、D1 数据、Workers 与 Queues 运维、KV 检查和 Token 权限放到一个本机界面，适合每天反复查看、复制、确认和排查的 Cloudflare 账号工作。

下载最新版 查看 GitHub 查看更新

产品定位

把 Cloudflare 账号里的日常动作收进一个可确认的工作台。

远程优先：R2、D1、Workers、Queues、KV、Token 权限与账号级状态，都围绕真实 Cloudflare 账号工作。

本地绑定仍交给官方 Local Explorer；CFDesk 负责远程资源、常用命令和高风险写操作前的确认。

功能范围

为查看、复制、发布和安全确认设计的入口。

01

R2 素材工作流

浏览存储桶，上传和下载文件，预览图片，复制公开 URL，检查公开域名；删除或覆盖对象前先确认。

02

D1 数据库工作

查看数据库和表，运行 SQL，查看结构图，管理索引，并导出常用格式。

03

Workers 运维

打开 Dashboard，检查设置和部署，查看近期健康信号，调整观测设置，复制常用 Wrangler 命令。

04

权限与安全操作

检查 Token 权限，对删除、覆盖、Secret、路由、域名、计划任务等远程写操作加入确认。

4 月 18 日之后

一个独立命名、独立发布的 Cloudflare 运维 fork。

Token 引导

通过 macOS Keychain 保存 API token，检查端点权限，并在 Wrangler 不可用时提供更清晰的回退路径。

R2 升级

加入缓存列表、图片预览、公开 URL 处理、传输设置、分片上传和更安全的变更确认。

Workers 和 Queues

加入 Workers 快捷操作、近期健康信号、指标、观测控制、设置检查，以及 Queue backlog 指标。

中英文界面

主要页面、设置页和资源操作文案进入中英文结构，方便后续持续维护。
